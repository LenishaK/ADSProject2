ADSProject2 - Music Playlist Manager 

##Overview 
This project is a C# console application that manages a music playlist using a custom linked list data structure.

The system allows users to add, delete, search, sort and play songs while demonstrating how linked lists work in practice.

---

## Features 
- Custom doubly linked list implementation (no built-in LinkedList)
- Load songs from a CSV file
- Add new songs to the playlist
- Delete songs by title or index
- Search songs by title or index (partial match supported)
- Display songs sorted by artist or duration
- Playback simulation ( next and previous wiwth wrap-around)
- Shuffle playlist randomly

---

## How to run 
1. open the project in visual studio
2. Ensure the CSV file is in the correct directory
3. Build and run the program
4. Use the menu to interaact with the playlist

---

##Example output 
Displays:
- Playlist of songs
- Playback navigation (current song, next, previous)
- Confirmation messages for add/delete actions
